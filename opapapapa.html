Aprenda JavaScript na prática com exemplos interativos

1. Variáveis e Tipos de Dados

Variáveis são espaços na memória onde armazenamos informações. Em JavaScript, temos diferentes tipos de dados: string (texto), number (números), boolean (verdadeiro/falso), entre outros.

// Declarando variáveis let nome = "João"; // String (texto) let idade = 25; // Number (número) let ativo = true; // Boolean (verdadeiro/falso) const PI = 3.14; // Constante (não muda)

2. Arrays (Listas)

Arrays são listas que armazenam múltiplos valores em uma única variável. Cada item tem uma posição (índice) começando do 0.

// Criando um array let frutas = ["Maçã", "Banana", "Laranja"]; console.log(frutas[0]); // Maçã (primeiro item) frutas.push("Uva"); // Adiciona no final

3. Operadores Matemáticos

Operadores matemáticos realizam cálculos: + (adição), - (subtração), * (multiplicação), / (divisão), % (resto da divisão).

let a = 10; let b = 3; console.log(a + b); // 13 (soma) console.log(a - b); // 7 (subtração) console.log(a * b); // 30 (multiplicação) console.log(a / b); // 3.33... (divisão) console.log(a % b); // 1 (resto)

4. Operadores Relacionais

Comparam valores e retornam true ou false: == (igual), != (diferente), > (maior), < (menor), >= (maior ou igual), <= (menor ou igual).

let x = 5; let y = 10; console.log(x == y); // false console.log(x < y); // true console.log(x >= 5); // true

5. Operadores Lógicos

Combinam condições: && (E - ambas verdadeiras), || (OU - pelo menos uma verdadeira), ! (NÃO - inverte o valor).

let temCNH = true; let maiorIdade = true; console.log(temCNH && maiorIdade); // true (ambas são true) console.log(temCNH || false); // true (uma é true) console.log(!temCNH); // false (inverte)

6. Estrutura Condicional IF/Else

Executa código baseado em condições. if executa se a condição for verdadeira, else executa se for falsa.

let idade = 18; if (idade >= 18) { console.log("Maior de idade"); } else { console.log("Menor de idade"); }

7. Estrutura de Seleção Switch/Case

Compara um valor com múltiplas opções. É útil quando há várias condições possíveis para verificar.

let dia = 1; switch(dia) { case 1: console.log("Domingo"); break; case 2: console.log("Segunda"); break; default: console.log("Outro dia"); }

8. Estrutura de Repetição While

Repete um bloco de código enquanto uma condição for verdadeira. Verifica a condição antes de executar.

let contador = 1; while (contador <= 5) { console.log("Contagem: " + contador); contador++; }

9. Estrutura de Repetição For

Repete um bloco de código um número específico de vezes. Ideal quando sabemos quantas repetições queremos.

for (let i = 0; i < 5; i++) { console.log("Iteração: " + i); }

💡 Tabuada:

10. Estrutura de Repetição Do While

Similar ao While, mas executa o código pelo menos uma vez antes de verificar a condição.

let numero = 1; do { console.log("Número: " + numero); numero++; } while (numero <= 3);

11. Estrutura de Repetição ForEach

Percorre cada elemento de um array executando uma função para cada item.

let frutas = ["Maçã", "Banana", "Laranja"]; frutas.forEach(function(fruta, indice) { console.log(indice + ": " + fruta); });

12. Funções em JavaScript

Funções são blocos de código reutilizáveis que executam uma tarefa específica. Podem receber parâmetros e retornar valores.

// Declarando uma função function somar(a, b) { return a + b; } // Chamando a função let resultado = somar(5, 3); console.log(resultado); // 8
